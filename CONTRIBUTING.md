# Contributing to AgentRoute Oracle

Thank you for your interest in contributing to AgentRoute Oracle! We welcome contributions from the community.

## How to Contribute

### Reporting Bugs

Found a bug? Please open an issue with:
- Clear description of the problem
- Steps to reproduce
- Expected vs actual behavior
- Your environment (OS, Python/Node version, etc.)

### Requesting Features

Have an idea? Open an issue with:
- Clear description of the feature
- Why it would be useful
- Potential implementation approach

### Submitting Code

1. **Fork the repository**
2. **Create a feature branch** (`git checkout -b feature/amazing-feature`)
3. **Make your changes**
4. **Write tests** if applicable
5. **Commit with clear messages** (`git commit -m 'Add amazing feature'`)
6. **Push to your fork** (`git push origin feature/amazing-feature`)
7. **Open a Pull Request**

### Areas We Need Help With

- **Routing algorithm improvements** - Optimize path finding
- **Client libraries** - Build clients in Go, Rust, etc.
- **Documentation** - Improve guides and examples
- **Integration examples** - Show how to use with popular frameworks
- **Bug fixes** - Help us squash bugs

## Code Style

- **Python:** Follow PEP 8
- **JavaScript:** Use Prettier formatting
- **Documentation:** Clear, concise, with examples

## Testing

Before submitting a PR:
- Run existing tests
- Add tests for new features
- Ensure all tests pass

## Community Guidelines

- Be respectful and inclusive
- Assume good intent
- Help others learn
- Share knowledge generously

## Questions?

- Open an issue for discussion
- Join our Discord community
- Email: support@agentroute.oracle
